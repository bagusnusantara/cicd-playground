## Cara Menggunakan Lab Ini
1. Clone repository ini.
2. Buka folder `docs/` dan ikuti panduan sesuai levelnya mulai dari `level-1-docker.md`.
3. Setiap level memiliki target yang harus dicapai sebelum lanjut ke level berikutnya.

## Struktur Folder
- `/app`: Source code aplikasi utama.
- `/docs`: Panduan langkah-demi-langkah.
- `Dockerfile`: Konfigurasi container.
- `Jenkinsfile`: Skrip pipeline (akan dikerjakan di Level 2+).
